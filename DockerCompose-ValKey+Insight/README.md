# Docker-Container für ValKey # 

<br>

Die [docker-compose.xml](docker-compose.xml)-Datei in diesem Verzeichnis
definiert zwei Container:
* ValKey-Server (In-Memory-Datenbank)
* Redis-Insight (Web-UI für Redis/ValKey)

<br>

Bei [ValKey](https://valkey.io) handelt es sich um einen Fork von Redis,
siehe z.B. [hier](https://www.heise.de/select/ct/2024/9/2409311024131767794).
